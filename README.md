# Rock Paper Scissors GUI

A simple Rock Paper Scissors game built with Python and Tkinter.

## Features

- Graphical UI with buttons for `Rock`, `Paper`, and `Scissors`
- Random computer move selection
- Scoreboard tracking:
  - Player wins
  - Computer wins
  - Ties
- Final score display when quitting

## Requirements

- Python 3.x
- Tkinter (usually included with Python on Windows)

## Running the game

1. Open a terminal in the project directory.
2. Run:

   ```bash
   python rock_paper.py
   ```

## Notes

- Click a move button to play a round.
- The score updates after each round.
- Click `Quit` to exit and view the final score.
